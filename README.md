# animation

## animation-fill-mode

forwards: 애니메이션 마지막 상태값 유지

backwards: 애니메이션 첫번째 상태값 유지

none: 애니메이션 끝나면 기본값유지

both: 애니메이션 시작전에도 첫번째 상태값, 끝난 후에도 마지막 상태값 유지

## matrix

## transform-box

## transform-origin
